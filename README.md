# Высотная программа полёта БПЛА

## Описание задания

Данный проект реализует консольный Python-модуль формирования **высотной программы полёта беспилотного летательного аппарата (БПЛА)** на основе профиля рельефа местности.
Высотная программа представляет собой список точек смены высоты вида:

```
(пройденное расстояние, высота полёта)
```

Между этими точками БПЛА движется горизонтально на постоянной высоте, а в самих точках выполняет строго вертикальный подъём или спуск.

### Входные данные

На вход подаётся:

* кусочно-линейный профиль рельефа вдоль маршрута;
* максимальное допустимое число точек смены высоты.

### Требования к формируемой программе

Сформированная высотная программа должна удовлетворять следующим условиям:

1. Высота полёта **никогда не ниже высоты рельефа**.
2. Отсутствуют избыточные точки смены высоты (высота реально меняется).
3. Минимизируется интеграл квадрата отклонения высоты полёта от высоты рельефа
   (приближение к рельефу в среднеквадратичном смысле).
4. Количество точек смены высоты не превышает заданного максимума.

Проект ориентирован на практическое применение и может служить основой для дальнейших улучшений алгоритма (оптимизация, учёт динамических ограничений, расширение модели движения).

## Формат входного и выходного файлов

### Входной файл

1. **Первая строка** — два целых числа:

   * `M` — максимальное число точек смены высоты;
   * `N` — число точек профиля рельефа.
2. **Вторая строка** — `N` пар вещественных чисел:

   ```
   distance_0 height_0 distance_1 height_1 ... distance_N height_N
   ```

### Выходной файл

1. **Первая строка** — фактическое число точек смены высоты.
2. **Вторая строка** — список точек смены высоты в том же формате:

   ```
   distance height distance height ...
   ```

## Запуск скрипта

Скрипт является консольным приложением и принимает два аргумента командной строки.

### Запуск

```bash
python flight_programm.py input.txt output.txt --plot
```

где:

* `input.txt` — путь к входному файлу с профилем рельефа;
* `output.txt` — путь к файлу, в который будет записана высотная программа;
* `--plot` — опциональный флаг для отображения графика полета.

### Требования

* Python 3.8+
* matplotlib 3.10.7 (опционально для отображения графика)
